# Contributing

## Development

```bash
npm install
npm run dev
```

## Code Style

- Use TypeScript strict mode
- Run `npm run lint` before committing
- Run `npm run typecheck` before committing

## Testing

```bash
npm test
```

## Pull Requests

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Submit a PR

## Issues

Open an issue for bugs or feature requests.
