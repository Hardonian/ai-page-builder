# AI Page Builder

**AI-powered link-in-bio store builder with auto-layout from products.**

## What it does

- Upload product photos → AI generates optimized store layout
- One-click publish to link-in-bio
- A/B testing suggestions
- Built on FlexibleAccessible components

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Claude API / OpenAI
- FlexibleAccessible components
